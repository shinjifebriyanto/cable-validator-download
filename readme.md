# ⚡ Cable Validator App

**Cable Validator** adalah aplikasi mobile Android yang dirancang khusus untuk mengukur, memantau, dan memvalidasi hantaran resistansi konduktor kabel berdasarkan standar internasional (SNI dan ASTM AWG) secara presisi melalui koneksi Bluetooth Low Energy (BLE).

## ✨ Fitur Utama

- **🔗 Perangkat (Pindai & Hubungkan):** Integrasi nirkabel menggunakan teknologi BLE untuk menghubungkan *smartphone* dengan instrumen keras, memastikan transmisi data sensor yang hemat daya dan stabil.
- **⚙️ Konfigurasi Parameter:** Modul pengaturan variabel pengujian secara presisi, meliputi standar acuan (SNI/ASTM AWG), ukuran penampang kabel, panjang kabel, dan persentase toleransi.
- **📊 Validasi Real-time:** Menampilkan grafik pembacaan nilai resistansi secara langsung dan melakukan komparasi otomatis dengan batas standar untuk menentukan status kelayakan akhir (**LULUS** atau **GAGAL**).
- **📂 Riwayat Pengujian:** Sistem pencatatan lokal (*local storage*) terintegrasi yang menyimpan rekam jejak seluruh hasil pengujian beserta detail parameter waktunya.
- **📖 Panduan Rujukan:** Pusat dokumentasi bawaan yang menyajikan tabel standar pengujian kabel internasional dan referensi kalkulasi rumus resistansi teoritis.
